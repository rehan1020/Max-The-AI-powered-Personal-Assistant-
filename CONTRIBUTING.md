# Contributing to Max

Thank you for your interest in improving Max! We welcome contributions of all kinds.

## How to Contribute

### 🐛 Reporting Bugs
If you find a bug, please open a GitHub Issue with:
- A clear title and description.
- Steps to reproduce the issue.
- Your Windows version and Python version.
- Any relevant log files from the `logs/` directory.

### ✨ Requesting Features
We love new ideas! Explain the feature you want and how it would improve the user experience.

### 💻 Pull Requests
1. **Fork the repository** on GitHub.
2. **Create a branch** for your feature or fix.
3. **Write clean code** and follow the existing style.
4. **Test your changes** carefully using the test suite.
5. **Submit a Pull Request** with a detailed description of your changes.

---

## Development Setup

1. Install development dependencies:
   ```powershell
   pip install pytest
   ```
2. Run tests:
   ```powershell
   pytest
   ```
