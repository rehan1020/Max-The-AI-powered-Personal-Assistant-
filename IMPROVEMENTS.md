# 🚀 Recent Improvements

This document tracks major updates and features added to Max.

## [v0.2.6] - 2026-03-29
### ✨ Added
- **Experimental Multi-Platform Support**: Initial readiness for **Linux** and **macOS**.
- **OS-Specific Requirement Files**: New `requirements-linux.txt` and `requirements-macos.txt` for easier setup on non-Windows systems.
- **Universal Documentation**: Complete refresh of all guides to be platform-neutral.
- **Beta Notices**: Added status indicators and troubleshooting for experimental platforms.

## [v0.2.5] - 2026-03-29
### ✨ Added
- **New Logo**: Professional branding integrated into the repository and UI.
- **Advanced System Controls**: Direct commands for WiFi, Bluetooth, Sleep, and Lock.
- **Clipboard Sync**: New actions to copy/paste/clear the Windows clipboard via voice.
- **Battery Status**: Check your system's battery percentage and charging state.
- **GitHub-Optimized Docs**: Full overhaul of `README.md`, `COMMANDS.md`, and `SAFETY_ARCHITECTURE.md`.
- **Standard Repo Files**: Added `LICENSE` (MIT) and `CONTRIBUTING.md`.

### ⚡ Optimized
- **Rule Parser**: Expanded pattern matching for instant, non-LLM responses.
- **Architecture**: Refined the `RuleParser` -> `LLM` -> `Validator` pipeline for better speed and safety.

## [v0.2.0] - 2026-03-27
### ✨ Added
- **Automatic Fallback**: Seamless transitions between Ollama (local) and OpenRouter (cloud).
- **GUI Refresh**: Dark-themed PyQt6 window with real-time status bars and conversation logs.
- **Screen Reading**: Integrated OCR (EasyOCR) for visual multitasking.
